# TD4 GraphQL - Bruno

Importez ce dossier dans Bruno.

## Environnement
Ouvrez l'environnement `Local` et renseignez :
- `baseUrl` : http://localhost:8055
- `token` : votre token Directus (role td4_reader ou admin)

Les requetes d'authentification utilisent `{{token}}` sauf la requete `Auth-no-token`.

## Contenu
- `Q1` a `Q10` : requetes GraphQL du TD4
- `Auth-*` : tests d'acces avec/sans token
- `M1` a `M8` : mutations demandees
